# Core Pillars

## Pillar 1: Active Naval Combat Is The Main Game

Sunken Fortune must be built around active combat, not passive income. The player should progress fastest by actively fighting, choosing targets, controlling position, and using ship builds intelligently.

Combat should reward:

- Movement.
- Positioning.
- Broadside angles.
- Reload timing.
- Ammunition choice.
- Cannon layout.
- Defensive modules.
- Enemy pattern recognition.
- Boss mechanic execution.

## Pillar 2: Ships Are Emotional Milestones

The game should include roughly 7-8 major ships over the long term. Each ship should feel like a meaningful identity upgrade with distinct role, silhouette, combat profile, and progression context.

Ships should not become a list of 15+ disposable purchases. The player should remember when they earned a new vessel.

Potential ship differentiation:

- Fast raider.
- Balanced brig.
- Heavy broadside ship.
- Durable warship.
- Cargo-capable treasure hunter.
- Specialist boss hunter.
- Late-game flagship.

## Pillar 3: Progression Has Multiple Meaningful Paths

Progression should combine player skill, build planning, and long-term investment.

Supported progression sources:

- Gold economy.
- Ship upgrades.
- Cannon upgrades.
- Ammunition tiers.
- Modules.
- Talents.
- Captain upgrades.
- Crew upgrades.
- Crafting.
- Zone unlocks.
- Boss drops.
- Rare materials.

## Pillar 4: Offline Progress Supports, But Never Replaces, Play

Autonomous Voyage allows the player's crew to continue sailing while the player is away. It should provide capped, reduced rewards and strengthen retention without becoming the optimal way to play.

Design target:

| Activity | Target Reward Relationship |
| --- | --- |
| 1 focused hour of active combat | Roughly equivalent to a full 14-hour Autonomous Voyage window |
| 14-hour Autonomous Voyage | Useful background progress, capped and incomplete |

## Pillar 5: Economy Integrity Matters

The game includes progression currencies, premium currency simulation, crafting materials, and offline earnings. Save reliability, auditability, and tunable values are core design needs, not afterthoughts.

Economy rules should be:

- Tunable.
- Logged where important.
- Resistant to accidental duplication.
- Clear to players.
- Balanced around active combat first.

## Pillar 6: Originality Is A Production Constraint

Sunken Fortune can be inspired by genre patterns, but it must not copy names, art, UI layouts, maps, factions, quests, dialogue, music, icons, item names, lore, or exact creative expression from existing games.

Originality should be reviewed continuously, not only at launch.
