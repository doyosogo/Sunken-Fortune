# Game Vision

## Vision Statement

Sunken Fortune is a PvE-first pirate naval action RPG where active combat, ship mastery, captain progression, and long-term treasure hunting create the feeling of building a feared pirate legacy across dangerous seas.

The game should feel approachable like a browser RPG, but deeper and more skillful than an idle economy game. The best rewards should come from playing well, making smart build choices, and overcoming increasingly dangerous enemies.

## Player Fantasy

The player is a pirate captain building a reputation from a modest vessel into a feared command. Progression should feel earned through battle, treasure recovery, crew improvement, and ship upgrades.

Key fantasy beats:

- Commanding a ship in tactical naval combat.
- Timing broadsides and reloads under pressure.
- Returning to port with damaged hull, loot, and stories.
- Unlocking a new ship as a major milestone.
- Preparing for harder waters with better cannons, modules, crew, and ammunition.
- Reading a Captain's Log after time away and seeing what the crew accomplished.

## Tone

Sunken Fortune should be adventurous, dangerous, and treasure-driven. It may include humor and character, but it should not become parody-first. The world should support mystery, risk, rival captains, cursed waters, lost wrecks, smuggling routes, naval powers, and strange sea legends.

## Experience Goals

| Goal | Description |
| --- | --- |
| Combat-first progression | Active play is the fastest and most rewarding path. |
| Meaningful upgrades | Each upgrade should change capability, strategy, or aspiration. |
| Ship milestones | Ships are major identity moments, not disposable stat sticks. |
| Fair offline progress | Autonomous Voyage helps retention without replacing play. |
| Legally original identity | The game must build its own names, world, UI, art, lore, and systems expression. |
| Tunable economy | Values should be adjustable as balance data improves. |

## Target Player

Sunken Fortune is for players who enjoy:

- Naval combat and pirate themes.
- Long-term RPG progression.
- Browser MMO-style structure.
- Loot, upgrades, crafting, and bosses.
- Session-based combat with some offline continuity.
- Optional premium currency systems that do not erase free progression.

## Design North Star

When systems conflict, prioritize:

1. Active combat quality.
2. Progression integrity.
3. Legal originality.
4. Long-term economy health.
5. Clear player feedback.
